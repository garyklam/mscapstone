Only image files with the following extensions will be accepted: .jpg, .jpeg, .png, and .gif
The most likely prediction is placed highest with confidence decreasing with each result. The confidence is listed as a percentage next to the species name of each prediction.
ShroomGuesser only contains the data for 45 species. If your image is not one of these, a low confidence is likely. Additionally, if your image does not capture the identifying features of the species, the prediction model may have a difficult time with identification.
The technical terms are used to make similar but different features distinct from each other and to help familiarize the user with such terms. For more info on these terms and helpful diagrams, check out this useful website.
For a basic description of certain terms, look below:
